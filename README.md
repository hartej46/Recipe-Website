# 🍽️ Meal Master (Recipe Finder)

Meal Master is a premium, high-performance recipe discovery web app designed for modern kitchens. Whether you're a seasoned chef or just starting out, Meal Master helps you find incredible dishes based on what's in your pantry.

URL;- https://meal-master-recipe-finder.vercel.app/

---

## 🚀 Key Features

### 🔍 Intelligence Search
- **Dynamic Discovery**: Search by dish name or specific ingredients.
- **Cuisine Filters**: Instantly sort results by cultural origin (Italian, Indian, Chinese, etc.).
- **Live Search**: Rapid results with persistent state management.

### 🍱 Detailed Culinary Guides
- **Visual Ingredients**: View ingredient images for quick recognition.
- **Step-by-Step Instructions**: Clear, numbered directions for every recipe.
- **Multimedia Integration**: Direct links to YouTube video tutorials.
- **Meta Tags**: Categorical tags (Vegan, Side, Dessert) for quick context.

### 🌓 Premium Dark Mode
- **Native Implementation**: High-performance "Pure Black" theme toggler.
- **Persistent State**: Theme preference is saved locally for consistent user experience.
- **Glassmorphic Design**: Modern, responsive UI with localized CSS overrides for zero-flicker transitions.

### 📈 Smart Suggestions
- **Chef's Special**: Dynamically generated featured recipe on every visit.
- **Most Loved & trending**: Randomized recommendations to keep your menu fresh.
- **"You Might Also Crave"**: Contextual suggestions on recipe detail pages.

---

## 🛠️ Tech Stack

- **Frontend**: Vanilla JavaScript (ES6+), HTML5
- **Styling**: Premium Vanilla CSS (Modular & Localized Overrides)
- **API**: [TheMealDB API](https://www.themealdb.com/api.php)
- **Persistence**: LocalStorage for theme and state management

---

## 📖 How it Works
1. **Search**: Enter a keyword in the landing page search bar.
2. **Filter**: Refine your results using the dynamic cuisine buttons on the results page.
3. **Cook**: Select a dish to see its full profile, watch the video, and start cooking!
4. **Theme**: Use the toggle in the nav bar to switch between Light and a high-contrast Dark Mode.
